Project Overview

Hackonauts is an AI-driven chatbot web application designed to analyze social media datasets and
provide insights such as trend identification and user engagement metrics. Developed as a pre-
hackathon project for the Level Supermind Hackathon, the application leverages advanced
technologies to assist users in understanding and interpreting social media data effectively.

The chatbot interface allows users to interactively query social media data, receiving real-time
analytics and visualizations. By integrating AI capabilities, Hackonauts can process large volumes of
data to extract meaningful patterns and trends, aiding in strategic decision-making for social media
campaigns.

The development focused on creating a user-friendly and responsive interface using Streamlit,
ensuring accessibility across various devices. The backend integrates Langflow for workflow
automation, DataStax for scalable data storage, and the OpenAI API for natural language processing,
enabling the chatbot to understand and respond to user queries effectively.
