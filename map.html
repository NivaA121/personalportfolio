Project Overview

PDEU Navigation is a web-based application designed to enhance campus navigation for students,
faculty, and visitors at Pandit Deendayal Energy University (PDEU). The application allows users to
scan QR codes placed at various campus locations to determine their current position and select a
destination, providing step-by-step directions to their chosen endpoint.

The primary goal of the project was to simplify the navigation experience within the university
premises, especially for newcomers and guests. By integrating QR code scanning with an interactive
map interface, the application offers a user-friendly solution to locate departments, administrative
offices, lecture halls, and other facilities efficiently.

The development process emphasized creating a responsive and intuitive user interface compatible
with various devices. The application leverages web technologies to deliver real-time navigation
assistance without the need for additional installations, ensuring accessibility and ease of use for all
campus visitors.
